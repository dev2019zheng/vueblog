# Manjaro-kde 安装微信

吐槽: 最近刚装的manjaro，结果kubuntu上的deepin-wine-ubuntu一套用不了了，上网上百度，都是一些抄来抄去的答案，基本都是 deepin-wechat的错误包名， 以及electronic-wechat，虽然能用，但不是我要的啊，最后不死心试了一下kubuntu上的安装包名：deepin-wine-wechat， 结果神奇的能用了！

本文为原创博文,禁止转载,如需转载请联系作者:鱼知鱼

# 重点写在前面，安装命令

yaourt -S deepin-wine-wechat

随手附tim安装

yaourt -S deepin-wine-tim
